# NOIR Scripts - Guiones Sci-Fi Profesionales

Colección de herramientas y recursos para la generación de guiones de ciencia ficción de calidad profesional.

## 📁 Estructura del Repositorio

```
verbose-memory/
├── pages/
│   └── landing-guiones.html       # Landing page principal de NOIR Scripts
├── generators/
│   ├── ScriptGenerator-Pro.html    # Generador con biblioteca pre-cargada (v1)
│   ├── ScriptGenerator-Pro-2.html  # Versión mejorada (v2)
│   └── scriptwriter-sci-fi.html    # Generador con API (v3)
└── README.md
```

## 🎬 Páginas y Herramientas

### Landing Page - NOIR Scripts
**Archivo**: `pages/landing-guiones.html`

Página de presentación profesional con:
- Branding futuristico (Cyberpunk)
- Portfolio de 6 guiones principales (Cyberpunk, Space Opera, Distópico, Post-Humano, Contacto Alien, Viajes en Tiempo)
- Opciones de pricing (Individual, Pack 3, Studio 6)
- Formulario de contacto
- Diseño responsivo con animaciones

**Géneros disponibles**:
- 🎯 Cyberpunk
- 🚀 Space Opera
- 🏚️ Distópico
- 🧠 Post-Humano
- 👽 Contacto Alien
- ⏰ Viajes en Tiempo

### Script Generator Pro v1
**Archivo**: `generators/ScriptGenerator-Pro.html`

Generador con biblioteca pre-cargada de guiones:
- Interfaz React moderna
- 2 géneros (Cyberpunk, Space Opera)
- 2 duraciones (3 y 5 minutos)
- 4 guiones profesionales pre-escritos
- Funciones de copiar y descargar

### Script Generator Pro v2
**Archivo**: `generators/ScriptGenerator-Pro-2.html`

Versión mejorada con más contenido:
- 6 géneros disponibles
- 3 duraciones (3, 5, 10 minutos)
- 18 guiones profesionales en la biblioteca
- Interfaz optimizada
- Mejor manejo de estado

### Script Writer - Sci-Fi v3
**Archivo**: `generators/scriptwriter-sci-fi.html`

Generador avanzado con integración potencial a API:
- Soporte para duración flexible (3, 5, 10, 15, 30 minutos)
- 6 géneros sci-fi
- Sistema de prompts personalizados
- Estructura lista para integración con Claude API
- Funciones de copiar y descargar

## 🎯 Usos

1. **Landing Page**: Presentar los servicios de NOIR Scripts
2. **Generadores**: Crear guiones profesionales de ciencia ficción
3. **Portfolio**: Mostrar ejemplos de guiones en diferentes géneros

## 📊 Contenido de Guiones

Cada archivo contiene:
- Sluglines (INT./EXT., LOCATION, TIME)
- Descripciones de acción
- Diálogos naturales y impactantes
- Acotaciones técnicas
- Formato screenplay profesional

## 🎨 Diseño

- **Color Scheme Cyberpunk**:
  - Colores primarios: Verde neón (#00ff41), Púrpura (#ff006e)
  - Fondo: Gradientes oscuros y profundos
  - Tipografía: Orbitron (futurístico) + Crimson Text (elegancia)

- **Componentes**:
  - Tarjetas con efectos hover
  - Animaciones de deslizamiento
  - Efectos de brillo (glow)
  - Diseño responsivo (mobile-first)

## 🚀 Características Técnicas

- HTML5 semántico
- CSS3 moderno (Grid, Flexbox, animaciones)
- React 18 (en generadores)
- Sin dependencias externas (excepto fuentes y React)
- Totalmente funcional offline
